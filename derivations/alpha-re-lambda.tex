\section{Derivation of $\alpha = r_e / \bar{\lambda}$}

\subsection{Definitions}

The fine-structure constant $\alpha$ is defined as:
\begin{equation}
\alpha = \frac{e^2}{4\pi\varepsilon_0 \hbar c}
\end{equation}

The classical electron radius $r_e$ is defined as the radius at which
the electrostatic self-energy of a uniformly charged sphere equals
the electron's rest mass energy:
\begin{equation}
r_e = \frac{e^2}{4\pi\varepsilon_0 m_e c^2}
\end{equation}

The reduced Compton wavelength $\bar{\lambda}_e$ (also denoted \lambdabar) is:
\begin{equation}
\bar{\lambda}_e = \frac{\hbar}{m_e c}
\end{equation}

\subsection{Derivation}

Divide $r_e$ by $\bar{\lambda}_e$:
\begin{align}
\frac{r_e}{\bar{\lambda}_e}
&= \frac{ \dfrac{e^2}{4\pi\varepsilon_0 m_e c^2} }
        { \dfrac{\hbar}{m_e c} } \\[12pt]
&= \frac{e^2}{4\pi\varepsilon_0 m_e c^2} \cdot \frac{m_e c}{\hbar} \\[12pt]
&= \frac{e^2}{4\pi\varepsilon_0 \hbar c} \\[6pt]
&= \alpha
\end{align}

Therefore:
\begin{equation}
\boxed{\alpha = \frac{r_e}{\bar{\lambda}_e}}
\end{equation}

\subsection{Equivalent Forms}

Using the full Compton wavelength $\lambda_e = h/(m_e c) = 2\pi\bar{\lambda}_e$:
\begin{equation}
\alpha = \frac{2\pi r_e}{\lambda_e}
\end{equation}

Using the Bohr radius $a_0 = 4\pi\varepsilon_0\hbar^2/(m_e e^2) = \bar{\lambda}_e/\alpha$:
\begin{equation}
\alpha = \frac{\bar{\lambda}_e}{a_0}
\end{equation}

\subsection{The Full Scaling Chain}

\begin{align}
r_e &= \alpha \bar{\lambda}_e = \alpha^2 a_0 \\
\bar{\lambda}_e &= \alpha a_0 \\
a_0 &= r_e / \alpha^2
\end{align}

All three intrinsic electron length scales are locked together by powers of $\alpha$.
